## Описание программы 
*"GitHub User Finder — это tkinter приложение для поиска пользователей на GitHub,позволяет добовлять в избранное пользователей и переходить на их GitHub. "* --- 
## Требования для запуска 
Что нужно установить на компьютер: - Python 3.10 или выше - Библиотеки: `pip install matplotlib requests` --- 
## Как запустить 
```bash 
git clone https://github.com/andr752/GitHub-User-Finder.git
